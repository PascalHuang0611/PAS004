# 水池系統比較 — 各系統最優解

資料來源:`reports/` 內全部 882 份模擬報表(7 系統 × 2 玩家配置 × 7 組 Buffer 區間 × 3 種預墊比例 × 3 次獨立 Run)。
每個系統先以三次 Run 的平均值評估其 21 種參數組合,挑出該系統自己的最優參數,再將各系統的最優解並列於下表。

**本文件為盲測評比**:七個系統僅以代號 B~H 呈現,不說明各系統的內部機制,也不做推薦結論。
由於每個系統有 21 種參數組合、全部看完負擔太大,下表的用途是告訴你**每個系統優先看哪一組參數**——
請在 `index.html` 儀表板上切到該系統的最優參數組合,搭配分布圖與本表指標,自行判斷哪個系統最符合需求。

## 最優解的挑選方式

1. **硬門檻**:整體 RTP 落在 99~101%、跨 BET 之 RTP 差距 < 3%、三次 Run 間 RTP 波動 < 1.5%。
2. **體感評分**(通過門檻者中取分數最低):開局第 5/10/20 局勝率貼近 50%、連輸 ≥8 與 ≥10 局的玩家占比越低越好、首勝出現越早越好、10 局未贏比例越低越好、P(贏｜前一局贏) 越接近 50% 越好。

## 指標解讀

- **勝率@第N局 / 累計RTP@第20局**:玩家入場初期的體感,越貼近 50% / 100% 代表開局不冷場也不放水。
- **連輸≥N局玩家占比**:體感下限。作為參照,一顆 50% 的公平硬幣在 200 局中約有 28% 玩家會經歷 8 連輸,這是隨機的數學本質;明顯低於此值代表系統有在控制連輸。
- **P(贏｜前一局贏)**:輸贏的自相關。50% 代表無規律可循;明顯高於 50% 代表輸贏成串(連莊感),明顯低於 50% 代表贏完容易輸(節奏感),兩者長期都可能被玩家察覺。
- **玩家間 RTP 標準差 / 最低最高玩家 RTP**:玩家之間的貧富差距。越大越有大起大落的刺激感,越小則人人結果趨同。
